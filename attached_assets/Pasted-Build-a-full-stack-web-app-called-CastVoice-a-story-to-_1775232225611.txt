Build a full-stack web app called "CastVoice" — a story-to-audio-drama engine where users cast characters with real or AI-generated voices and get a fully produced audio drama with sound effects and emotional delivery.

=== TECH STACK ===
- Frontend: React + Vite + TypeScript
- Styling: Tailwind CSS
- Animations: Framer Motion
- Backend: Node.js + Express
- Database: Replit DB (built-in)
- Auth: Replit Auth (built-in)
- APIs: ElevenLabs (Voice Design, Voice Cloning, TTS, Sound Effects), Pollinations.ai (free image generation, no key needed)

=== DESIGN DIRECTION ===
Dark cinematic theme. Think film production studio meets audio drama. Color palette: deep black (#0a0a0a) background, warm gold (#f5a623) as primary accent, off-white (#f0ece4) for text. Font pairing: "Playfair Display" for headings (dramatic, editorial), "DM Sans" for body (clean, readable). Every page should feel like entering a recording studio. Grain texture overlay on backgrounds. Subtle glowing effects on interactive elements. No purple gradients. No generic AI aesthetics.

--- PAGE 6: GENERATION SCREEN ---
Full screen cinematic loading experience. Do NOT use a plain spinner.

Show:
- Story title at top
- Scene background image (blurred, atmospheric)
- Progress steps with animated indicators:
  Step 1: "Designing character voices..." 
  Step 2: "Writing emotional delivery for each line..."
  Step 3: "Generating dialogue..."
  Step 4: "Adding sound effects..."
  Step 5: "Mixing the final drama..."
- Each step lights up gold as it completes
- Estimated time: "This takes 3-5 minutes. Worth it."
- Ambient audio waveform animation underneath

Backend generation flow (run sequentially):
1. For each character with Voice Design option: call ElevenLabs Voice Design API with the description text
2. For each character with Voice Clone option: use stored clone_id
3. For each line of dialogue in the script:
   - Determine emotion tag from script data
   - Call ElevenLabs TTS API with voice_id + text + emotion context in the voice settings
   - Use ElevenLabs voice settings: stability, similarity_boost, style tuned per emotion
4. For each SFX cue in the script: call ElevenLabs Sound Effects API with the cue description
5. Mix all audio files in sequence using Node.js (use fluent-ffmpeg or similar)
6. Generate scene image for each scene via Pollinations.ai API: fetch("https://image.pollinations.ai/prompt/{encoded_prompt}?width=1280&height=720&nologo=true")
7. Store final mixed audio file and scene images, update story status to "ready"

=== ELEVENLABS API CALLS ===

Voice Design:
POST https://api.elevenlabs.io/v1/voice-generation/generate-voice
Body: { "text": "voice description", "gender": "auto", "age": "auto", "accent": "auto" }

Voice Cloning:
POST https://api.elevenlabs.io/v1/voices/add
Form data: name, files (audio), description

TTS with emotion:
POST https://api.elevenlabs.io/v1/text-to-speech/{voice_id}
Body: { "text": "dialogue line", "model_id": "eleven_multilingual_v2", "voice_settings": { "stability": 0.3, "similarity_boost": 0.8, "style": 0.7, "use_speaker_boost": true } }
Adjust stability per emotion: panic = 0.1, calm = 0.7, sad = 0.5, excited = 0.2

Sound Effects:
POST https://api.elevenlabs.io/v1/sound-generation
Body: { "text": "SFX description from script", "duration_seconds": 3, "prompt_influence": 0.3 }
