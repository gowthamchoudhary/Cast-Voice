Voice Design API call:
POST https://api.elevenlabs.io/v1/voice-generation/generate-voice
Headers: { "xi-api-key": process.env.ELEVENLABS_API_KEY }

  "text": "Hello, I am ready to perform."
}

=== TTS FOR EACH DIALOGUE LINE ===
POST https://api.elevenlabs.io/v1/text-to-speech/{voice_id}
Headers: { 
  "xi-api-key": process.env.ELEVENLABS_API_KEY,
  "Accept": "audio/mpeg"
}
Body: {
  "text": "{dialogue text}",
  "model_id": "eleven_multilingual_v2",
  "voice_settings": {
    "stability": {line.stability or 0.5},
    "similarity_boost": 0.85,
    "style": {
      panic: 0.9,
      excited: 0.8, 
      sad: 0.6,
      calm: 0.2,
      nervous: 0.7,
      default: 0.5
    }[line.emotion] or 0.5,
    "use_speaker_boost": true
  }
}
Response is raw audio/mpeg bytes.
Save to temp file: /tmp/line_{projectId}_{index}.mp3

=== SFX FOR EACH CUE ===
POST https://api.elevenlabs.io/v1/sound-generation
Headers: { "xi-api-key": process.env.ELEVENLABS_API_KEY }
Body: {
  "text": "{sfx_description from script}",
  "duration_seconds": 3,
  "prompt_influence": 0.3
}
Save to: /tmp/sfx_{projectId}_{scene}_{index}.mp3

=== FFMPEG MIX ===
Build concat list in correct order:
- SFX before scene (if exists)
- All dialogue lines for that scene in order
- Next scene SFX, then dialogue, etc.

Write /tmp/concat_{projectId}.txt:
file '/tmp/sfx_...'
file '/tmp/line_...'
...

Run ffmpeg:
ffmpeg -f concat -safe 0 -i /tmp/concat_{projectId}.txt \
  -c:a libmp3lame -q:a 2 \
  /tmp/final_{projectId}.mp3

Store final file, serve at GET /api/audio/{projectId}
Stream with proper range request support for browser playback.
Clean up all temp files after storing final.
